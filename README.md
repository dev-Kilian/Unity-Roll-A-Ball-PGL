# Unity-Roll-A-Ball-PGL
 Simple 3D game to learn the basics of Unity.

What's this game about? 🚀
This project is a 3D platform game where you play as a Ball. Your objective is to collect all the yellow items on the map, while avoiding the enemies you find throughout the experience.

Specs recommended 📋
Any modern PC with recent versions of Windows will be able to run this game. Testing for other platforms (such as Mac or Linux) is yet to be performed.

How to install 🔧
This game comes in a portable, .exe file. This file can be found inside the root folder of the project, in a folder called 'Builds'.
IMPORTANT NOTE: The name of the exectuable file is 'prueba-pgl-2-roll-a-ball.exe'.

How to play ⚙️
Use the WASD or the directional keys to control the Ball's movement.
Use the Spacebar to jump.

Collect all the yellow items before getting caught by the enemy to win the game.
You lose the game by allowing the enemy to make contact with the Ball.

After ending a level (either winning or losing), the game will advance or reset the level, according to the results. This will happen automatically after a few seconds.

Major game issues 🔩

- During some point in the development, the enemy stopped chasing the player directly and instead started avoiding him, making it really difficult for the player to lose the game.
